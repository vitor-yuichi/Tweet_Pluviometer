\chapter{Objetivos}\label{secObjetivos}

%RGN: Acredito que seja necessário omitir esse vinculo com o cemaden, pois não existe discussão anterior que deixaria isso claro. Ainda, acho que essa info não seja relevante para a Fapesp.

A finalidade deste projeto é o desenvolvimento de um modelo de Aprendizado de Máquina para a predição de alagamentos, utilizando-se dos diferentes algoritmos propostos pela pesquisa. Neste projeto será utilizado séries temporais da rede social Twitter, a base de dados de alagamentos em São Paulo, e dados pluviométricos e meteorológicos.

%RGN: incluir mais objetivos especificos: publicar artigo, construir e disponibilizar um banco de dados sobre alagamentos e dados de tweet, disponibilizar os códigos em repositório publico, [veja o exemplo do Felipe]
Objetivos específicos: 
\begin{enumerate}
    \item Definir qual é o algoritmo mais preciso para relacionar os dados meteorológicos, pluviométricos e frequência de tweets, para o modelo de classificação de alagamentos em dias de "ocorrência" e "não-ocorrência".
    
    \item Estabelecer o atributo com maior relevância estatística para predição de alagamentos dentre os propostos, combinando-os de forma conjunta com os algoritmos. 
    
    \item Publicação dos resultados em congressos, eventos e periódicos científicos.
    
    \item Estruturar e disponibilizar as bases de dados e códigos em um repositório público. 
\end{enumerate}
