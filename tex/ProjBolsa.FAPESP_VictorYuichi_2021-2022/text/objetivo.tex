Este projeto de pesquisa tem como objetivo principal o desenvolvimento de um método para previsão de alagamentos com base em técnicas de Aprendizado de Máquina, dados temporais extraídos de Tweets e instrumentos meteorológicos como radares e pluviômetros.

Como objetivos específicos, são elencados: 
\begin{enumerate}
    \item Analisar e definir o algoritmo de Aprendizado de Máquina de melhor desempenho para previsão de alagamentos a partir de dados meteorológicos e extraídos de Tweets; %, para o modelo de classificação de alagamentos em dias de "ocorrência" e "não-ocorrência".
    
    \item Investigar sobre os atributo de maior relevância para predição de alagamentos; % dentre os propostos, combinando-os de forma conjunta com os algoritmos. 

    \item Estruturar e disponibilizar as bases de dados e códigos em repositório público;
    
    \item Publicação dos resultados obtidos em eventos e periódicos científicos.
    

\end{enumerate}
