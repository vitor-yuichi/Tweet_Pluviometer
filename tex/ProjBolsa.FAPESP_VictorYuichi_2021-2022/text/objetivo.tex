Este projeto de pesquisa tem como objetivo principal o desenvolvimento de um método para previsão de alagamentos com base em técnicas de Aprendizado de Máquina, dados temporais extraídos de Tweets, radar meteorológico e pluviômetros.

% modelo de Aprendizado de Máquina para a predição de alagamentos, utilizando-se dos diferentes algoritmos propostos pela pesquisa. Neste projeto será utilizado séries temporais da rede social Twitter, a base de dados de alagamentos em São Paulo, e dados pluviométricos e meteorológicos.

%RGN: incluir mais objetivos especificos: publicar artigo, construir e disponibilizar um banco de dados sobre alagamentos e dados de tweet, disponibilizar os códigos em repositório publico, [veja o exemplo do Felipe]
Como objetivos específicos, são elencados: 
\begin{enumerate}
    \item Analisar e definir o algoritmo de Aprendizado de Máquina de melhor desempenho para previsão de alagamentos a partir dos dados extraídos de Tweets e de instrumentos meteorológicos (radar meteorológico e pluviômetros); %, para o modelo de classificação de alagamentos em dias de "ocorrência" e "não-ocorrência".
    
    \item Investigar os atributo (i.e., dados de Tweets, radar meteorológico, pluviômetros e/ou derivados) de maior relevância para predição de alagamentos; % dentre os propostos, combinando-os de forma conjunta com os algoritmos. 

    \item Estruturar e disponibilizar em repositório público as bases de dados utilizadas e códigos-fonte implementados na pesquisa;
    
    \item Divulgar os resultados obtidos em eventos e periódicos científicos.
    

\end{enumerate}
